# GATO FELICIO FELPUCIO AMBULANTE

Projeto voltado para fins educacionais, com intenção de aprender javascript, focando nos conceitos da condicional if ternario, list comprehension e funções.


## Aprendizados

Compreendi os conceitos da condicional if ternario, list comprehension e funções.

## Funcionalidades

- Movimentação com as teclas de seta,
- cura, dano, loja e ataque com os botôes.

## Stacks utilizada

**Front-end:** Html, css & JavaScript.



## Deploy

Para fazer o deploy desse projeto faça um clone do repositório e execute o arquivo em index.html.

```bash
  git clone https://github.com/MateusMengardaMcM/Gato-Felicio-Ambulante
```
